# Accessibility Standard: Metric Handbook Accessibility

## Details from Metric Handbook, Page 69

• IPMS 1, IPMS 2 and IPMS 3 need to be measured and stated 
separately. To ensure transparency, marked up plans should 
accompany measurement reports, which should be detailed to 
include the breakdown of the measure; this should provide ben-
efit to the end user.
