# Accessibility Standard: Metric Handbook Accessibility

## Details from Metric Handbook, Page 59

3.3 Wheelchair-user dimensions 
These dimensions exclude the wheelchair user and only consider 
length and width; see Figure 4.4. Dimensions in Table III are given 
for occupied and unoccupied wheelchairs over a range of wheel-
chair types including scooters.
