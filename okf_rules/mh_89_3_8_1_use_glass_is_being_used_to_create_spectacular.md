# Accessibility Standard: Metric Handbook Accessibility

## Details from Metric Handbook, Page 89

8.1 Use 
Glass is being used to create spectacular structures in modern 
architecture and it can be incorporated as a structural material in a 
number of ways. Structural sealant glazing is described in BS EN 
13022: 2014 Parts 1 and 2.
