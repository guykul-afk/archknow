# Accessibility Standard: Metric Handbook Accessibility

## Details from Metric Handbook, Page 81

• Simply supported. The beam ends are assumed to be completely 
free to move rotationally resulting in zero moment at each end. 
In addition, one end is assumed to be supported on a roller bear-
ing allowing freedom of movement in the direction of the beam

length resulting in zero horizontal reaction at the roller bearing 
end. If the loading is uniformly distributed along the length of 
the beam then the bending moment will be a maximum at mid-
span. This type of beam is known as statically determinate as the 
support reactions and bending moments can be calculated using 
the basic equations of equilibrium.
