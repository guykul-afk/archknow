# Accessibility Standard: Metric Handbook Accessibility

## Details from Metric Handbook, Page 82

4.3 Vertical load-bearing elements 
A wall is a vertical load-carrying element whose length in plan is at 
least four times its width; otherwise, it is a column. A pier is a col-
umn integral with a wall. References to walls apply also to columns 
and piers unless stated otherwise.
