# Accessibility Standard: Metric Handbook Accessibility

## Details from Metric Handbook, Page 81

• Cantilever. This beam type is supported only at one end where 
it is fixed in both position and rotationally. The bending moment 
will be zero at the free end and a maximum at the support caus-
ing tension in the top of the beam.
