# Accessibility Standard: Metric Handbook Accessibility

## Details from Metric Handbook, Page 61

4.5 Changes in level 
Changes in level cause problems for many people, particularly peo-
ple with mobility impairments or people who are blind or partially 
sighted. Even a single step can prevent access for someone with 
mobility impairment and can present a trip hazard for all people. 
Any gradient less steep than 1:60 can be considered level. 
Where changes in level cannot be avoided, slopes with as shallow 
a gradient as possible (less steep than 1:21) should be provided as 
it is preferable to avoid the use of ramps (a slope with a gradient of 
1:20 or steeper).
