# Accessibility Standard: Metric Handbook Accessibility

## Details from Metric Handbook, Page 59

3.4 Pushchair dimensions 
The length and width dimensions of child pushchairs are highly 
variable. There is not, as is the case with manual wheelchairs, a 
standard model for informing design criteria. Figure 4.5 gives com-
mon dimensions.
